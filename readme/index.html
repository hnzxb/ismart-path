ismart-path

nodejs path组件扩展

- path.smartFilepath.smartFile()
自动创建文件 

path.smartFile("./temp","./log","server.log") =>  {root}/temp/log/server.log

- path.smartDirpath.smartDir()
自动创建文件夹 

path.smartFile("./temp","./log","2018/10") =>  {root}/temp/log/2018/10
